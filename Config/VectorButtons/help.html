Ctrl+Enter -- Draw (works only when cursor is in the textarea)

Strict input is equal to FVWM spec. Light is a non-standard and designed only for easy typing.

A minimum example of light input: 

0 0 1, 100 100 0
This will draw a single diagonal line from top left corner to bottom right. The FVWM equivalent is: 
Vector 2 0x0@1 100x100@0
If X or Y coordinate appears to be < 0 or > 100, it will be substituted with 0, so be attentive.
